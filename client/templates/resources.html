Project Technology Stack

Every component of this site is documented as a learning exercise in building scalable, secure cloud applications.

Backend & Deployment

  • **Infrastructure as Code:** Terraform (used for AWS RDS, EC2, ECR, VPC).
  • **CI/CD:** GitHub Actions (custom SSH deployment workflow).
  • **Compute:** AWS EC2 (for host) running Docker Compose.
  • **Containerization:** Docker / NGINX (as a reverse proxy).
  • **Secrets Management:** AWS Secrets Manager (used for DB credentials).

Application Stack

  • **Language:** Python 3.11
  • **Framework:** Flask
  • **Database:** PostgreSQL (AWS RDS)
  • **Authentication:** Flask-Login, Flask-Bcrypt (Secure hashing).
  • **Styling:** Tailwind CSS (built into the Docker image via multi-stage build).
